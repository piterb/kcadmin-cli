# kcadmin-cli

Standalone npm CLI package for kcadmin operations.

## Requirements

- Node.js >= 20
- npm >= 10

## Install (local dev)

```bash
npm install
npm test
```

## Run CLI

```bash
node ./bin/kcadmin.js
# or after global install / npx:
kcadmin
```

## Publish flow

```bash
npm login
npm whoami
npm pack
npm publish --access public
```
